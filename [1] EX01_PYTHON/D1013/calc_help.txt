명령어 도움말:
  값 변경/조회
    set <x>           : 현재 값을 x로 설정
    clear             : 현재 값 0으로 초기화
    result            : 현재 값 출력

  이항 연산 (좌항=현재 값)
    add <x>           : 현재 값 += x
    sub <x>           : 현재 값 -= x
    mul <x>           : 현재 값 *= x
    div <x>           : 현재 값 /= x

  기록
    hist [k]          : 최근 k개 기록(기본 10)

  기타
    help              : 도움말
    x | exit | quit   : 종료
