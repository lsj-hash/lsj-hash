수집된 데이터 정리 => 전처리

1) 빈칸 / 데이터 없는 것 검사 => 결측치 Missing value
    => 재수집
    => 일부 미미하게 존재 ==> 처리 : 버리기 / 치환

2) 중복 데이터 검사
    => 날씨 데이터 년 월 일 시 온도 습도 풍속 => 검사 -> 유지
    => 학생 데이터 학년 학번 학교 이름 성별 => 검사 -> 버리기

3) 이상한 값 즉, 이상치/특이값 검사
    => 컬럼별로 유효한 값의 범위
    => (예 : 성별 - 남, 여 / 나이 - 1, 130)
    => 검사/처리 -> 삭제/치환
    => 데이터가 가진 특성 체크 꼭!!!
